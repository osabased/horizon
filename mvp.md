# Horizon MVP

## Goal

Build the smallest useful Horizon that proves the core loop:

```text
Evidence -> Context Pack -> OpenCode -> Observation -> Better Next Pack
```

The MVP should not try to implement the full vision. It should prove that Horizon can make agentic development cheaper, faster, and less error-prone on real projects by giving OpenCode better context and better validation guidance.

## Non-goals

- Do not build a new agent.
- Do not build a new IDE.
- Do not replace OpenCode.
- Do not build a general memory
